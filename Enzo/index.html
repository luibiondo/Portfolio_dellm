About Me

Turning raw data into trusted, actionable datasets

I'm a Data Engineer from Buenos Aires, Argentina, specializing in building scalable data pipelines, cloud data infrastructure, and real-time streaming systems. I enjoy transforming raw, messy data into clean, reliable datasets that power business decisions.

My focus is designing and implementing robust ETL/ELT pipelines, data warehouses, and data lake architectures that handle terabytes of data efficiently. I work extensively with tools like Apache Spark, Apache Kafka, dbt, Airflow, and cloud platforms such as AWS, GCP, and Snowflake. I'm passionate about data quality, observability, and building systems that teams can trust.
